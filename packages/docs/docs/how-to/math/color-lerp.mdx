# Tween colors in different color spaces

`Color.createLerp()` builds a custom interpolation function so you can tween
colors through whichever color space gives the result you want:

```tsx
yield * node().fill('#4aaaf1', 2, easeInOutCubic, Color.createLerp('hsl'));
```

The supported color spaces are `lch` (the default), `rgb`, `hsl`, `hsv`, `hsi`,
`lab`, `hcl`, and `lrgb`. Colors in Canvas Commons are powered by
[chroma.js](https://gka.github.io/chroma.js/), which is where these spaces come
from.

## Which space to pick

The default `lch` works well for most transitions. Use `lab` when you want
perceptually uniform steps (no unexpected brightness dips mid-tween). `hsl`
rotates through the hue wheel, which can produce rainbow artifacts on long
transitions. `rgb` is the fastest but often looks muddy when the two colors are
far apart.
